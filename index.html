Layout

Most layout is handled automatically, using left and right padding to set smart defaults of usable interior width.

two columns

Add two columns to an element to make children elements display in two columns.

INPUT

<div class="two columns">
  <div class="light blue">California</div>
  <div class="light blue">Stylesheets</div>
</div>

RESULT

California
Stylesheets

three columns

Add three to make children elements display in three columns. This works up to twelve columns, which is the default.

INPUT

<div class="three columns">
  <div class="light blue">California</div>
  <div class="light blue">Stylesheets</div>
  <div class="light blue">(CASS)</div>
</div>

RESULT

California
Stylesheets
(CASS)

golden columns

Use the golden class with columns to get two columns in the golden ratio.

INPUT

<div class="golden columns">
  <div class="light blue">California</div>
  <div class="light blue">Stylesheets</div>
</div>

RESULT

California
Stylesheets

  <div class="light blue">Stylesheets</div>
</div>

wide

padding

Use the class padding to pad color edges. The classes padding-top, padding-right, padding-bottom, and padding-left work too.

Add tight for tighter padding.

INPUT

<div class="dark blue padding">dark blue</div>

RESULT

dark blue

INPUT

<div class="blue tight padding">blue</div>

RESULT

blue

Typography

CASS enforces a sensible page width and text measure (how long the words run in a line) automatically.

type, card, page and full

The width classes let you adjust the default width. They work by default on an inner-width approach. You can add the class width to give it a more traditional width.

This paragraph is a padding red card paragraph.

This paragraph is a padding red card width paragraph.

Adjust the --text-measure CSS custom property to adjust text measure. Tweak the default line height and font size with the --line-height and --font-size-base CSS custom properties. Set --text-indent to 0 to disable the automatic text indent. Or just use the no-text-indent class.


text-color

Use text-color with a color to color text instead of the background color.

INPUT

<h2 class="text-color red">A red headline</h2>

RESULT

A red headline

Responsiveness

CASS has three breakpoint targets: mobile, tablet and desktop. These classes work with the columns, fixed and hide classes, enabling these behaviors only where you want them.

mobile

Only on mobile

tablet

On tablet and above

desktop

On desktop

mobile tablet

On mobile and tablet

CASS has different settings for each target, with font size and padding being much smaller on mobile.

INPUT

<div class="two columns tablet">
  <div class="light blue">California</div>
  <div class="light blue">Stylesheets</div>
</div>

INPUT

<span class="hide mobile">hide on mobile</span>

INPUT

<div class="fixed desktop">fixed on desktop</div>
