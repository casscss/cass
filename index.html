The easiest CSS system.

CASS does what the big frameworks do... without all the framework.

This page uses a California Stylesheet, the no-workflow, no-code, custom-property-powered modern CSS file that works like a framework.

  • Typography-first
  • Hundreds of colors
  • No build step
  • Just 8kb
  • No abbreviations or numerals
  • Only normal English words (and some from California)

No workflow

No gzipping. No tree-shaking. No stress. Just use the CSS.

Great for docs

Give your documentation a great set of type standards and colors in one small file.

Write less CSS

Write 90% fewer lines of CSS with smart defaults that anyone can learn and use.

Philosophy

Use CASS to quickly frame out your pages into usable, testable layouts. CASS is designed to meet 90% of your CSS needs with just 8kb of CSS.

Spend your CSS-writing time on that last 10% that really sets your page apart, not on repeating the basics of typography, layout, color and responsiveness.

Layout

Most layout is handled automatically, using left and right padding to set smart defaults of usable interior width.

two columns

Add two columns to an element to make children elements display in two columns.

INPUT

<div class="two columns">
  <div class="light blue">California</div>
  <div class="light blue">Stylesheets</div>
</div>

RESULT

California
Stylesheets

three columns

Add three to make children elements display in three columns. This works up to twelve columns, which is the default.

INPUT

<div class="three columns">
  <div class="light blue">California</div>
  <div class="light blue">Stylesheets</div>
  <div class="light blue">(CASS)</div>
</div>

RESULT

California
Stylesheets
(CASS)

padding

Use the class padding to pad color edges. The classes padding-top, padding-right, padding-bottom, and padding-left work too.

Add tight for tighter padding.

INPUT

<div class="dark blue padding">dark blue</div>

RESULT

dark blue

INPUT

<div class="blue tight padding">blue</div>

RESULT

blue

Typography

CASS enforces a sensible page width and text measure (how long the words run in a line) automatically.

Adjust the --text-measure CSS custom property to adjust text measure. Tweak the default line height and font size with the --line-height and --font-size-base CSS custom properties. Set --text-indent to 0 to disable the automatic text indent.

Most typography features are automatic. You can use the type class to enforce text measure inside an element, or the card class for a smaller, card-sized text measure.


text-color

Use text-color with a color to color text instead of the background color.

INPUT

<h2 class="text-color red">A red headline</h2>

RESULT

A red headline

Responsiveness

CASS has three breakpoint targets: mobile, tablet and desktop. These classes work with the columns, fixed and hide classes, enabling these behaviors only where you want them.

mobile

Only on mobile

tablet

On tablet and above

desktop

On desktop

mobile tablet

On mobile and tablet

CASS has different settings for each target, with font size and padding being much smaller on mobile.

INPUT

<div class="columns tablet">
  <div class="light blue">California</div>
  <div class="light blue">Stylesheets</div>
</div>

INPUT

<span class="hide mobile">hide on mobile</span>

INPUT

<div class="fixed desktop">fixed on desktop</div>
