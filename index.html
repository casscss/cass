California Stylesheets CASS
 

It's like CSS, but easier.

California Stylesheets are all you could ever need from a CSS framework... without the framework.

This page uses CASS, the no-workflow, no-CSS, custom-property-powered modern CSS file that works like a framework. CASS lets you style pages fast without actually writing CSS.

Since every California Stylesheet is a CSS file, there's nothing to install, run, preprocess or compile.

Which projects are good for CASS?

Tiny sites will benefit from the full feature set. Easily add updates/features/content when you come back and make an update months later.

Razor-thin CSS files will leave you writing more CSS later. California Stylesheets let you express your design in the HTML.

Medium-sized sites are a great match for CASS, since it acts like a central repository of design information for a project.

The California Stylesheets paradigm lets you fashion your website on-the-fly to adjust to changing requirements. It's as flexible as HTML and CSS because it is HTML and CSS.

Huge, complicated apps are the best for California Stylesheets. The flexibility of the no-workflow approach lets you spin up standards-powered projects quickly, without tons of administrative overhead for something as simple as CSS.

More people on the project means more people who save time by not tangling with CSS.
