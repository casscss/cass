Get your California Stylesheet:

The easiest CSS system.

All you could ever need from a CSS framework... without the framework.

This page uses California Stylesheets, the no-workflow, no-CSS, custom-property-powered modern CSS file that works like a framework. CASS lets you style pages fast without actually writing CSS.

Since every California Stylesheet is a CSS file, there's nothing to install, run, preprocess or compile. Just reference the file or throw it in a style tag and the conventions will be available on your page.

No workflow

No gzipping. No tree-shaking. No stress. Simply use the CSS file.

Upgrade-friendly

Swap in a new California Stylesheet when you rebrand or update your site.

Anyone can read it

Simple class names, taken mostly from CSS, with no abbreviations or numbers.

Composables are CSS classes or groups of classes which give stylistic meaning to an HTML element:

<button class="big dark red">
  A button
</button>

Composables work with any HTML element that's visible on the page.
