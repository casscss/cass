Like CSS, but easier.

Easy to write and easy to read, California Stylesheets are all you could ever need from a CSS framework... without the framework.

This page is using CASS, the no-workflow, no-CSS, custom-property-powered modern CSS file that works like a framework. Use it to write styles in HTML.

Since it's just a CSS file, there's nothing to install or run. Grab it here or customize your own below.

California Stylesheets is a CSS paradigm for devs and teams who want to spend way less time on CSS

No workflow

No gzipping. No tree-shaking. No stress. Simply use the CSS file.

Upgrade-friendly

Swap in a new California Stylesheet when you rebrand or update your site.

Anyone can read it

Simple class names, taken mostly from CSS, with no abbreviations or numbers.

California Stylesheets in a nutshell

TL;DR: Everything is a CSS class that describes HTML elements. To get a red div with standard text inner-width and padding:

<div class="text inner-width padding red"></div>

Use tint (for white text) and shade (for black text) to override text color, as adverbs and helpers don't enforce text contrast standards.

Helpers, variants and adverbs can all work together on a single color, making for more than 1200 color combinations.

By default, color classes change the background color of an element. When text-color is added, they instead change the text color.

The text width class represents how wide your text is supposed to be.

Use text inner-width to enforce text measure inside an element. Use page inner-width for layout container elements.

Don't forget to use margin-auto to center elements where necessary.

Which projects are good for CASS?

Tiny sites will benefit from the full feature set. Easily add updates/features/content when you come back and make an update months later.

Razor-thin CSS files will leave you writing more CSS later. California Stylesheets let you express your design in the HTML.

Medium-sized sites are a great match for CASS, since it acts like a central repository of design information for a project.

The California Stylesheets paradigm lets you fashion your website on-the-fly to adjust to changing requirements. It's as flexible as HTML and CSS because it is HTML and CSS.

Huge, complicated apps are the best for California Stylesheets. The flexibility of the no-workflow approach lets you spin up standards-powered projects quickly, without tons of administrative overhead for something as simple as CSS.

More people on the project means more people who save time by not tangling with CSS.
