Widths

Classes for setting widths

Widths

Width classes set the width of an element.

  • .type width: The base content width, based on the @text-measure variable.
  • .card width: A smaller supplemental width, golden to type width
  • .page width: The page width, like type width and card width added together

These classes are often used for the basic layout framing of a page. They work well with margin-auto, which will center them in their parent container.

Inner widths

Inner-width classes use padding to emulate an inner-width effect.

  • .text inner-width: Enforces text measure inside an element
  • .card inner-width: Enforces the smaller text measure/width of an aside
  • .page inner-width: On very large elements, sets the inner width to the full page width

Inner-width composables work with the grid composables. This column of text, for example, has the text inner-width class on a nine wide element. In cases where the grid width goes above the text measure, the text inner-width class will enforce it.

Inner-width classes help cut down on markup complexity by letting you use fewer containers.
