Settings

Adjust the look and feel of your project with CASS's CSS custom properties.

CSS Custom properties

Your project's settings are at the top of your ca.css file.

Here's everything you'll find there.

  • --line-height: Sets the project's default line height
  • --text-measure: How man characters are on a line (very approximate)
  • --font-size-base: The base font size on desktop
  • --font-size-mod: The default font-size multiplier
  • --font-family: The default font-family
  • --headline-font-family: The default headline font-family
  • --headline-base: The base headline size on desktop
  • --headline-base-mod: The default headline font-size multiplier
  • --text-padding: The value used by text-padding
  • --text-margin: The value used by text-margin
  • --text-indent: The amount to indent paragraphs
  • --text-border-radius: The radius of bordered text
  • --monospace: The monospace font stack
  • --serif: The serif font stack
  • --sans-serif: The sans-serif font stack
  • --padding-base: The project's base padding
  • --font-size: An element's font-size
  • --responsive-width: The value used for responsive width
  • --padding: An element's padding
  • --margin: The value used by margin
  • --gutter: The value used by gutter
  • --gap: The default gap in grids
  • --width: The default width value
  • --text-width: The value used by text width and inner text width
  • --card-width: The value used by card width and inner card width
  • --page-width: The value used by page width and inner page width
  • --full-width: The value used by full width
  • --multi-column: The default mult-column scheme
  • --one-column: The default one-column scheme
  • --contrast: How much light is lightened by
  • --h: The hue value of the current color
  • --s: The saturation value of the current color
  • --l: The lightness value of the current color
  • --o: The overlay alpha value
  • --warm: The color used by the warm helper
  • --cool: The color used by the cool helper
  • --dull: The gray used by the dull helper
  • --border-width: The default border width
  • --border-style: The default border style
  • --border-radius: The default border radiuss
  • --border-h: The default border's hue
  • --border-s: The default border's saturation
  • --border-l: The default border's lightness
  • --border-color: The default border's color
  • --border: The default border
  • --border-thin: The value used by thin
  • --border-thick: The value used by thick
  • --border-radius: The default border-radius
  • --box-shadow: The default shadow
  • --shadow-position: The default shadow position
  • --shadow-blur: The default shadow blur
  • --shadow-color: The default shadow color

Theming

You can create a theme by overwriting the CSS custom properties and including it on your page after ca.css

:root {
    --font-size-base: 18px;
    --line-height: 2px;
    --text-measure: 58;
    --padding-base: .5rem;
}

This CASS theme makes for a more academic look by only changing a few custom properties. Any of the variables in the prior section can be overwritten in this style.

You can also just edit the variables that come with your CASS file directly, if that's easier. Just remember that when you swap in a new one, you'll be potentially losing those old settings.

Unfortunately, changing any of the colors here won't change the automatic color palette generation. You should get a new CASS if you want a new color scheme.
